## 만약 실패하면 (rollback 방법)

```powershell
# 잘못된 commit 을 만들었다면 — 마지막 커밋 취소 (변경사항은 유지)
git reset --soft HEAD~1

# 잘못된 tag 를 push 했다면 — local 과 origin 양쪽에서 삭제
git tag -d v0.6.0
git push origin :refs/tags/v0.6.0

# Streamlit Cloud 가 빨간불이면 — 이전 commit 으로 revert
git revert HEAD
git push origin main
```

MCPize/Glama 제출 실패 — submit 페이지에서 "Withdraw submission" 클릭. 24시간 후 재제출 가능.
